# 行终止符

**what**：
文本中一行的结束和新行的开始
1. `Unix`：`LF`
2. `Windows`：`LFCR`（源于早期打字机逻辑）

 >  **LF (Line Feed, `\n`)**：将纸张向上卷一行（换行）
 
>  **CR (Carriage Return, `\r`)**：将打印头移回行首（回车）

| 操作系统                     | 行终止符    | 转义字符   | 英文名称            |
| :----------------------- | :------ | :----- | :-------------- |
| **Unix / Linux / macOS** | 换行      | `\n`   | `LF` (**最佳实践**) |
| **Windows / DOS**        | 回车 + 换行 | `\r\n` | `CRLF`          |
