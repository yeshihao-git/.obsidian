# 2 地址划分
## 2.1 传统ABC类地址

**what：**
将 IP 地址分为 网络号+主机号
- A类 前8位为网络号， 其中前缀1位值为0  ，用于区分网络类型
  0  （1位） 网络号（ 7位） 主机号（24位）

- B类 前16位为网络号，其中前缀2位值为10 ，用于区分网络类型
  10 （2位） 网络号（14位） 主机号（16位）

- C类 前24位为网络号，其中前缀3位值为110，用于区分网络类型
  110（3位） 网络号（21位） 主机号（ 8位）

- D类 前缀4位值为1110，其他未定义

![[Pasted image 20260519182429.png|337]]

**划分子网方式** ：取部分主机号作为子网号。具体做法：采用 子网掩码 区分出子网号

```
B类网络 ：172.16.0.0，前16位为网络号
子网掩码：255.255.192.0，这里划分出的网络号为18，因此后面的2位为子网号
```

## 2.2 无分类地址CIDR

**what：**
a.b.c.d/x 前x位为网络号，剩下的为主机号

**why：** 
解决传统ABC类地址数量不够的问题

**划分子网方式** ：天生具备

### 2.2.1 子网

**what：**
是局域网的内部划分

```
某公司局域网 的 IP池 为 192.168.0.0/16
子网A：192.168.1.0/24
子网B：192.168.2.0/24
```
