# 等值线

**what**：
数值相等的点连起来的线。

**示例**：
- 地图上**海拔高度相同**的线 → 等高线
- 温度图上**温度相同**的线 → 等温线
- 压力场里**压力相同**的线 → 等压线
- vtkCutter 切出来的横截面

# 等值面

**what**：
空间中数值相等的点连起来形成的面。把等值线从 2D 扩展到 3D 空间，就成了等值面。

**示例**：
- 3D 温度场里，所有**温度=50℃**的点连起来 → 一个封闭曲面
- 3D 压力场里，所有**压力=100Pa**的点连起来 → 一个曲面
- 医学 CT 里，所有**密度相同**的点连起来 → 器官轮廓

# 剖切

# 切面

**what**：
用平面切 3D 模型，得到一个 2D 截面。

# 图元

**what**：

- 点
- 线
- 面

# 几何碰撞

| 术语       | 核心定义                    |
| -------- | ----------------------- |
| 间隙干涉     | 未发生穿透，但间距小于设定允许阈值       |
| 接触干涉     | 物体表面临界贴合，无间隙、无穿透        |
| 相交干涉（侵入） | 模型之间互相穿插、穿透             |
| 超界       | 物体运动范围超出限定空间边界          |
| 包络检查     | 校验物体运动包络范围，排查侵入、超界、干涉问题 |

# PMI 标注

**what**：
PMI标注（Product Manufacturing Information，产品制造信息）：将二维工程图上的尺寸、公差、粗糙度、基准、技术要求等制造信息，标注在**三维模型表面/空间**

**why**：
- 传统：**3D模型 + 一堆2D图纸**（人看图猜3D，易看错、漏看）
- PMI：**3D模型本身 = 图纸**（尺寸公差直接标在3D上，所见即所得，可直接传给机床/质检设备）
