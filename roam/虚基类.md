# 虚基类
**what**：
派生类通过 `virtual` 关键字继承基类，该基类为虚基类
1. **底层**：每个虚继承类中有 `vbptr` （虚基类指针），指向 虚基类表，虚基类表中记录 Base的偏移量，通过偏移量定位到 Base
```cpp
class Base {
public:
  int data {12};
};

class A: virtual public Base { }; // 虚继承
class B: virtual public Base { };

class Derived: public A, public B { };
/*
Derived对象内存结构：
[ A::vbptr ]        -> 指向A的虚基类表（记录Base的偏移量）
[ A的成员变量 ]
[ B::vbptr ]        -> 指向B的虚基类表
[ B的成员变量 ]
[ 唯一Base子对象 ]   -> 被A和B共享
,*/
```
**访问流程**：
1. 通过 `A/B::vbptr` 找到虚基类表，获得 `Base` 的偏移量 `offset`
2. 访问 `Derived`对象地址 + `offset` 定位到 `Base::data`

**why**：
用于解决 “菱形继承” 问题
> **菱形继承**：B、C 继承 A，D 继承 B、C，导致 D 中包含 A 中的多份成员变量副本，引发成员变量 *访问二义性* 、 *内存浪费* 问题。使用虚基类后，D 中A的成员变量只有1份，问题解决。


## 最派生类负责初始化最基类
```cpp
#include <iostream>
#include <string>

using namespace std;

class Person {
public:
	string m_name;
	Person(string name):m_name(name) {}
};

class Teacher : virtual public Person {
public:
	int m_workerID;
	Teacher(string name, int workerID):Person(name), m_workerID(workerID) {}
};

class Student : virtual public Person {
public:
	int m_studentID;
	Student(string name, int studentID) : Person(name), m_studentID(studentID) {}
};

class TeachingAssistant : public Teacher, public Student {
public:
	TeachingAssistant(string name, int workerID, int studentID): Person(name),Teacher(name, workerID), Student(name, studentID){}
}; // 最派生类TeachingAssistant负责初始化最基类Person

int main() {
	TeachingAssistant ta = TeachingAssistant("yst", 0, 1);
	cout << ta.m_name << endl;                  // yst
	cout << "----------------" << endl;
	ta.Teacher::m_name = "new teacherName";
	cout << ta.Student::m_name << endl;         // new teacherName，说明ta中只有一个m_name
}
```
