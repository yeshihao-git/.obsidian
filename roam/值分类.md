# 值分类

**三种主要的值分类**：
1. `glvalue`（广义左值）：有身份，不可移动 
2. `prvalue`（纯右值）：无身份，可移动
3. `xvalue`（将亡值）：有身份，可移动
```cpp
// 整体关系框架
                expression
               /          \
          glvalue        rvalue
          /      \        /     \
     lvalue        xvalue       prvalue  
```

**左值和右值**：
`lvalue`（左值）：非`xvalue`的`glvalue`
`rvalue`（右值）：`prvalue`或`xvalue`
