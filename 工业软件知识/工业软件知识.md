# 等值线
**what**：
数值相等的点连起来的线。

---

**示例**：
- 地图上**海拔高度相同**的线 → 等高线
- 温度图上**温度相同**的线 → 等温线
- 压力场里**压力相同**的线 → 等压线
- vtkCutter 切出来的横截面

# 等值面
**what**：
空间中数值相等的点连起来形成的面。把等值线从 2D 扩展到 3D 空间，就成了等值面。

---

**示例**：
- 3D 温度场里，所有**温度=50℃**的点连起来 → 一个封闭曲面
- 3D 压力场里，所有**压力=100Pa**的点连起来 → 一个曲面
- 医学 CT 里，所有**密度相同**的点连起来 → 器官轮廓

# 剖切
**what**：
用平面切 3D 模型，得到其中一半 3D 模型。

# 切面
**what**：
用平面切 3D 模型，得到一个 2D 截面。

# 图元
**what**：
构成复杂三维物体的最基本几何单元。常见基础图元如下：
- 点
- 线
- 面
