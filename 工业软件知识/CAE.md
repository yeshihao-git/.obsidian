# 1. CAE 仿真三阶段

```
前处理 ───────► 求解 ───────► 后处理
"造模型"    "算力、跑仿真"   "可视化、看结果"
```

## 前处理 — 建立物理模型

**核心：把真实问题变成数学问题**

做三件事：
1. **几何**：定义零件的3D形状
2. **网格**：把连续几何切分成有限个小块（节点+单元），这是"有限元"的"有限"二字来源
3. **工况**：定义材料属性、施加载荷、设置约束边界条件、选择分析类型

## 求解器 — 数值计算

**核心：解方程组**

前处理完成后，求解器将物理模型转化为大规模线性/非线性方程组 **Ku = F**（刚度矩阵 × 位移 = 力），然后求解得到每个节点的位移。再由位移推导出应变、应力。

- 线性静力：一次求解
- 模态分析：求特征值
- 非线性/瞬态：逐步迭代

## 后处理 — 可视化

**核心：把数字变成可理解的图形**

求解器输出的是海量数值（百万节点的位移/应力），后处理将其转化为：
- 变形图（看结构怎么变形）
- 应力云图（看哪里最危险）
- 动画（看过程如何演变）

# CAE 可视化
## 体网格
（Volume Grid / vtkUnstructuredGrid） 

**what**：
包含所有 3D 实体单元 的完整网格

**why**：
- 计算用 ：求解器需要完整的3D网格来计算应力、应变
- 截面剖切 ：可以切一刀，看内部应力分布
- 体积渲染 ：显示材料内部的密度、温度等场量

## 表面网格
（Surface Grid / vtkPolyData）

**what**：
只取体网格的 外表面 ，把3D单元的面提取出来

**why**：
- 可视化用 ：3D渲染只需要表面，效率高
- 边线提取 ：从表面网格可以提取边界线（ vtkExtractEdges ）
- 轻量显示 ：比体网格数据量小得多

## 体网格和表面网格可以用来做什么？
### 1. 变形动画（最常用）
```
表面网格 + 位移场 → 变形动画
```
每个节点加上位移向量，表面网格就"动"起来了。

### 2. 应力云图
```
表面网格 + 应力标量场 → 颜色云图
```
把每个节点/单元的应力值映射为颜色，显示在表面网格上。

### 3. 截面视图
```
体网格 + 切割平面 → 内部应力分布
```
用平面切过体网格，显示切面上的应力。

### 4. 边线框架
```
表面网格 → vtkExtractEdges → 边线网格 → 线框显示
```
提取所有边，用线框模式显示结构骨架。

# 概念
## 应力云图
**what**：
用不同颜色映射模型上**各单元**的**应力值**，直观展示结构应力分布、大小的可视化结果

>**应力值**：单位面积上的内力大小，反映材料承受载荷时的受力程度

>**内力**：物体受外力作用时，内部产生的抵抗变形的相互作用力
